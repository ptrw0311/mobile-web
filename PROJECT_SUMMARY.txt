================================================================================
🚀 快速開始
================================================================================

1. 安裝依賴:
   npm install

2. 設定環境變數:
   cp .env.local.example .env.local
   # 編輯 .env.local 填入 POSTGRES_URL

3. 初始化資料庫:
   npm run db:push

4. 啟動開發服務器:
   npm run dev
   # 訪問 http://localhost:3000
